# THE RESTO - SASS
Projet de mise en page réalisé dans le cadre de ma formation Développeur Web.

Demo live
__Voir la page__

## Objectif
Créer une landing page thématique restaurant en intégrant SASS/SCSS avec une architecture de fichiers structurée.

## Compétences travaillées

* Architecture SCSS modulaire
* Mise en page avec Flex & Grid box
* Responsive design avec les media queries

## Stack technique
SCSS/SASS compilé via Dart Sass. HTML natif. Aucun framework CSS externe.
